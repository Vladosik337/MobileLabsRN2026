# Gesture Clicker - Лабораторна робота №3

## Інструкція запуску

### Кроки для запуску

1. **Клонування репозиторію:**
   ```bash
   git clone https://github.com/MobileLabsRN2026/lab3.git
   cd lab3/GestureClicker
   ```

2. **Встановлення залежностей:**
   ```bash
   npm install
   ```

3. **Запуск проекту:**
   ```bash
   npx expo start
   ```
---

## Опис реалізованого функціоналу

### 1. Головний екран (Home Screen)
- **Лічильник очок** - відображає поточний рахунок гравця
- **Інтерактивний об'єкт "TAP ME"** - круглий об'єкт, що реагує на різні жести
- **Інформаційна панель** - показує доступні жести та кількість очок за кожен

### 2. Реалізовані жести (react-native-gesture-handler)

| Жест | Опис | Очки |
|------|------|------|
| **Tap** | Коротке натискання | +1 |
| **Double Tap** | Подвійне натискання | +2 |
| **Long Press** | Утримання 3 секунди | +5 |
| **Pan (Drag)** | Перетягування об'єкта | - |
| **Fling Right** | Свайп вправо | +1-10 (випадково) |
| **Fling Left** | Свайп вліво | +1-10 (випадково) |
| **Pinch** | Масштабування | +3 |
| **Rotation** | Обертання двома пальцями | +4 |

### 3. Сторінка завдань (Challenges Screen)
Реалізовано 9 завдань:
- ✅ Зробити 10 кліків
- ✅ Зробити подвійний клік 5 разів
- ✅ Утримувати об'єкт 3 секунди
- ✅ Перетягнути об'єкт
- ✅ Зробити свайп вправо
- ✅ Зробити свайп вліво
- ✅ Змінити розмір об'єкта (Pinch)
- ✅ Отримати 100 очок
- ✅ Повернути об'єкт (власне завдання)

Кожне завдання має:
- Іконку та кольорову індикацію
- Прогрес-бар виконання
- Позначку про завершення

### 4. Сторінка налаштувань (Settings Screen)
- **Перемикання теми** (світла/темна)
- **Статистика** - загальний рахунок та виконані завдання
- **Скидання прогресу** - з підтвердженням
- **Інформація про додаток**

### 5. Навігація
- Використано `@react-navigation/native` та `@react-navigation/bottom-tabs`
- Нижня панель навігації з трьома вкладками:
  - Play (Головний екран)
  - Challenges (Завдання)
  - Settings (Налаштування)

### 6. Стилізація
- **Styled Components** - для стилізації компонентів
- **Підтримка світлої та темної теми** через контекст ThemeContext
- **Адаптивний дизайн** з анімаціями (react-native-reanimated)
- **Material Design іконки** (@expo/vector-icons)

### 7. Анімації
- Пульсуюча анімація при натисканні
- Збільшення при подвійному кліку
- Масштабування при long press
- Плавне переміщення при drag
- Анімація свайпу

## Висновки

У ході виконання лабораторної роботи №3 було:

1. **Вивчено роботу з жестами** - реалізовано 8 різних типів жестів (Tap, Double Tap, Long Press, Pan, Fling Left/Right, Pinch, Rotation) за допомогою бібліотеки `react-native-gesture-handler`.

2. **Опановано анімації** - використано `react-native-reanimated` для створення плавних анімацій при взаємодії користувача з об'єктом.

3. **Реалізовано систему досягнень** - створено 9 завдань з відстеженням прогресу та візуальною індикацією виконання.

4. **Застосовано сучасну стилізацію** - використано Styled Components підхід та реалізовано підтримку світлої і темної теми.

5. **Налаштовано навігацію** - створено навігацію між екранами за допомогою `@react-navigation/native` з нижньою панеллю вкладок.

6. **Використано контекст React** - для глобального управління станом гри та темою застосунку.
